Task App Overview

About the Task App

The Task App is a web application built using Node.js, Express, and PostgreSQL. It allows users to manage tasks effectively.

Database Schema

The database consists of two main tables:

  • Tasks Table:
    • taskid: Primary key, uniquely identifies each task.
    • description: Describes the task.
  • Users Table:
    • uid: Primary key, uniquely identifies each user.
    • username: User's name.
    • taskid: Foreign key, related to the task the user is assigned to.
    • user_status: Represents the status of the user's task.
